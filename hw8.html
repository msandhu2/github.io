2. I suggest that we leverage the skills I've learned in XML class to track our IS expenditures using XQuery to analyze the data contained in our AssetManager application. After all, AssetManager stores the data in a relational database and XQuery can be used to calculate the organizations spending for an entire year. We can beakdown the amount of money spent on monitors, computers, laptops, and other peripherals. As an example, the XQuery code below is used to list all books that cost less than $30 from the hw8.xml file.

xquery version "1.0"; for $b in doc("hw8.xml")/bookstore/book where $b/price < 30 return $b/title

Using XQuery, we can query AssetManager to see how many laptops, monitors, docking stations, Arubas, etc were deployed per region for the Work From Home project.
